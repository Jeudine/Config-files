= Network help

== Network service

=== Restart

To restart the network service:

[source, shell]
----
sudo systemctl restart networking.service
----

After restarting the network service, you have to bring the wanted network interface up, for example with the `wlxa00460111568` interface:

[source, shell]
----
sudo ifup wlxa00460111568
----

== wpa_supplicant service

=== Restart

To restart the wpa_supplicant service:

[source, shell]
----
sudo systemctl restart wpa_supplicant.service
----

=== Add a new network

Run:

[source, shell]
----
sudo wpa_cli
----

To load the current configuration:

[source, shell]
----
> reconfigure
----

To scan:

[source, shell]
----
> scan
> scan_results
----

Choose the interesting network and add it to the list with the following commands:

[source, shell]
----
> add_network
----

This command will return a number `<nb>`.

[source, shell]
----
> set_network <nb> ssid "<name of the network>"
> set_network <nb> psk "<password of the network>"
----

If you want to add an *id_str* to the network:

[source, shell]
----
> set_network <nb> id_str "<id of the network>"
----

The network is now ready to use, run:

[source, shell]
----
> enable_network <nb>
----

To save the config, run:

[source, shell]
----
> save config
----
